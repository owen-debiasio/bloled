# bloled
A pure OLED theme for Zed with blue text. Based on Yogi Amoled.

## Install/Upgrade

```
$ curl -fsSL https://raw.githubusercontent.com/owen-debiasio/bloled/main/install.sh | bash
```

## Uninstall

```
$ curl -fsSL https://raw.githubusercontent.com/owen-debiasio/bloled/main/uninstall.sh | bash
```
